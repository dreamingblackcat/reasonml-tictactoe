# ReasonML Tictactoe

This is an experimental project to play around with ReasonML and BuckleScript by trying to implement a CLI Tictactoe game.

# Build
```
npm run build
```

# Build + Watch

```
npm run start
```

# Run
```
npm run game
```


# Editor
If you use `vscode`, Press `Windows + Shift + B` it will build automatically
